What is Techahtlon '21?

Techathlon, an event organized by IPEC (Inderprastha Engineering College,Ghaziabad). It is a code sprint-like event in which computer programmers and others involved in software development, including graphic designers, interface designers, project managers and others, often including subject-matter-experts, collaborate intensively on software projects. It aims to ‘Inspiring Young Minds’, participants are encouraged to answer the given problem statements and if they do, they have the option to have their project judged with the possibility of moving on to the Tech World.
It is divided into two major events
